# Temperature-Converter

A simple and responsive **Temperature Converter** built using **HTML, CSS, and JavaScript**. It converts temperatures between **Celsius** and **Fahrenheit** with a modern and user-friendly interface.

## ✨ Features

- 🌡️ Celsius ↔ Fahrenheit conversion
- ⚡ Instant results
- 📱 Responsive design
- 🎨 Modern UI
- ✅ Input validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
